## 一开始的方案

将这样的表达式：

```scheme
(let fib-iter ((a 1) (b 0) (count 10))
    (if (= count 0)
        b
        (fib-iter (+ a b) 
                  a 
                  (- count 1))))
```

转换成这样的表达式：

```scheme
(begin
  (define fib-iter
    (lambda (a b count)
      (if (= count 0)
            b
            (fib-iter (+ a b) 
                  a 
                  (- count 1)))))
  (fib-iter 1 0 10))
```

即完成这样的转换：

```scheme
(let ⟨var⟩ ⟨bindings⟩ ⟨body⟩)
=>
(begin
  (define <var> (lambda <binding variables> <body>))
  (<var> <binding values>))
```

## 解决方案

在最外面套一层无参lambda。

即将这样的表达式：

```scheme
(let fib-iter ((a 1) (b 0) (count 10))
    (if (= count 0)
        b
        (fib-iter (+ a b) 
                  a 
                  (- count 1))))
```

转换成这样：

```scheme
((lambda ()
   (begin
     (define fib-iter
       (lambda (a b count)
         (if (= count 0)
             b
             (fib-iter (+ a b) 
                a 
                (- count 1)))))
     (fib-iter 1 0 10))))
```

即完成这样的转换：

```scheme
(let ⟨var⟩ ⟨bindings⟩ ⟨body⟩)
=>
((lambda ()
   (begin
     (define <var> (lambda <binding variables> <body>))
     (<var> <binding values>))))
```
